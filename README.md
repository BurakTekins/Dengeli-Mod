# Dengeli Mod

A mod for **Europa Universalis IV** (Steam Workshop ID `3742028664`).

- **Supported game version:** v1.37.*
- **Tags:** Balance, Missions & Decisions, Gameplay, National Ideas, Technologies, Trade, New Nations, Events

## Custom content (highlights)

- **Bayreuth (BYT)** rework — starting provinces, national ideas, a custom mission tree, and two formable nations:
  - **Rhineward** (Hesse + Thuringia, Protestant/Reformed → Millennialist Theocracy, Westphalian missions)
  - **Margravaria** (Vienna, Catholic → Crusader State, Austrian missions)
- **Crown of Isles (Trinacria)** — reworked ideas, island-themed flag, and a colonist→Socotra mechanic for Sicily.
- Various balance tweaks, an always-open Suez Canal, and more.

## Installation

Copy this folder into your EU4 mod directory (or subscribe via the Steam Workshop) and enable it in the launcher.
